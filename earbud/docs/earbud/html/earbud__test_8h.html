◆ appTestPrimaryAddressIsFromThisBoard()

bool appTestPrimaryAddressIsFromThisBoard ( void  )

Report if the primary bluetooth address originated with this board

When devices are paired one of the two addresses is selected as the primary address. This process is effectively random. This function indicates if the address chosen is that programmed into this device.

The log will contain the actual primary bluetooth address.

To find out if the device is using the primary address, use the test command appTestIsPrimary().

The Bluetooth Address can be retrieved by non test commands. Depending on the test automation it may not be possible to use that technique (shown here)

>>> prim_addr = apps1.fw.call.new("bdaddr")
>>> apps1.fw.call.appDeviceGetPrimaryBdAddr(prim_addr.address)
True
>>> prim_addr
|-bdaddr : struct
| |-uint32 lap : 0x0000ff0d
| |-uint8 uap : 0x5b
| |-uint16 nap : 0x0002
>>>
Returns
TRUE if the primary address is that originally assigned to this board, FALSE otherwise FALSE
