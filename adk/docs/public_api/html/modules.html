 HDMAComponent is used to decide when handover is required between primary and secondary earbuds. It currently uses placement of earbud (in case, in ear, out ear) and critical battery level for making a handover decision. Decision making based on battery level is configurable. The input will be sourced from the State Proxy module, which is responsible for communication between the buds. To source these inputs, HDMA will register a task with State Proxy during initialization
