◆ hfp_voice_source_routed

unsigned hfp_voice_source_routed

When HFP eSCO starts on primary, if secondary is not connected, the mirror profile will immediately call Telephony_NotifyCallAudioConnected, and set this flag. Then when secondary later connects, the primary knows not to re-connect the voice source. If primary and secondary are connected when HFP eSCO starts, mirror profile clears this flag and defers calling Telephony_NotifyCallAudioConnected until mirroring is active.
